Algal Bloom Forecasting

The WQI was incorporated into a data scraping code which can be used to read real-time water quality data from a webpage and calculate its respective algal bloom risk. However, the use of real-time data is dependent on the source webpage, as the code will only refresh when the webpage data is updated.
